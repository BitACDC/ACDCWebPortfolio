SapPIIre is a capstone project focused on securing and simplifying how personally identifiable information is shared with government electronic forms. The app securely stores a user's personal information and autofills electronic versions of CSWD application forms, removing the need to manually re-enter sensitive data across multiple forms.

Users scan a QR code in the connected web app to transfer selected PII directly into the e-form, with full control over exactly which fields are shared through a checkbox-based selection system. All data transfer is end-to-end encrypted using a hybrid cryptosystem combining AES and RSA, balancing the speed of symmetric encryption with the secure key exchange of asymmetric encryption.

Built with Flutter and Dart on the mobile front end, alongside HTML and PostgreSQL for the connected web and database layers, the project is currently under active development. As Capstone Leader and Lead Mobile Front-End UI/UX Developer, the role covers leading the project direction as well as designing and building the mobile user experience.
